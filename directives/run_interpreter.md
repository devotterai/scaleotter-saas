# Run Open Interpreter

## Goal
Execute complex, multi-step coding or system tasks autonomously using Open Interpreter.

## Inputs
- `prompt`: A natural language description of the task to perform (e.g., "Analyze the data.csv file and create a plot").

## Tools
- `execution/run_interpreter.py`

## Outputs
- Console output logs detailing the steps taken and any results.
- Any files created or modified by the interpreter as requested.

## Steps
1.  Run `python execution/run_interpreter.py "<prompt>"`.
2.  Monitor the output for completion or errors.
